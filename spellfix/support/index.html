Getting started

Installation & first launch

How do I install SpellFix?

Download SpellFix from the Mac App Store. After install, launch the app once — it will ask for accessibility permission, which is required so SpellFix can read your selection and replace it with the corrected text. No data leaves your computer; the permission is used only locally.

Which Macs are supported?

SpellFix runs on Apple Silicon Macs (M1, M2, M3, M4, and later) with macOS 26 or later. Apple Foundation Model — the default AI engine — requires Apple Silicon and is not available on Intel Macs.

Does SpellFix need an internet connection?

No. Once installed, SpellFix runs entirely on-device. You only need internet for the initial download from the App Store and to download additional AI models if you choose to.

Features

Correction, tones & translation

Does SpellFix work in every app?

Yes. The double-tap ⌥ shortcut is global: Mail, Notes, Safari, Pages, Word, Slack, Figma, Notion — anywhere you can select text.

Which AI models are used?

By default, Apple Foundation Model — preinstalled on Apple Silicon Macs running macOS 26. From Settings → Models you can also download Qwen 3.5, Gemma 4, SmolLM3 or other supported local models. All models run on your computer; none of them require an internet connection after download.

How do I add or remove tones?

Open Settings → Tones. You can add new tones with your own prompt, rename existing ones, reorder them, or remove the ones you don't use. Each tone gets its own button in the SpellFix popup.

How do I change the keyboard shortcut?

Open Settings → Shortcut. Pick any modifier or key combination — double-tap ⌥, ⌘+E, F19, anything you like. The new shortcut takes effect immediately.

Is my text sent anywhere?

No. No network request is made during correction, reshaping or translation. You can verify this with Little Snitch or any equivalent network monitor.

Purchase & license

Free, Pro, and refunds

What's the difference between Free and Pro?

All features are identical. The only difference: the free version appends a small "— Corrected by SpellFix" watermark to generated text. The first 7 days are on Pro automatically, watermark-free.

Is it really a one-time purchase?

Yes. One payment and SpellFix Pro is yours forever, on every Mac signed in with the same Apple ID. No subscription, no renewal, no resale.

I bought Pro but it isn't activated on this Mac.

Open SpellFix and go to Settings → Account → Restore purchase. Make sure you are signed in to the App Store with the Apple ID used for the original purchase. If the issue persists, email us with your purchase receipt number.

What is the refund policy?

Apple's App Store rules apply. You can request a refund directly from Apple within 14 days at reportaproblem.apple.com. We don't process payments ourselves, so refunds must go through Apple.

Troubleshooting

Something isn't working?

The double-tap shortcut doesn't trigger.

Most common cause: SpellFix doesn't have accessibility permission. Open System Settings → Privacy & Security → Accessibility and make sure SpellFix is listed and enabled. If it still doesn't work, try changing the shortcut in SpellFix Settings → Shortcut, then changing it back.

Correction is slow or unresponsive.

The first correction after launch loads the AI model into memory and can take a few seconds. Subsequent corrections are instant. If you experience persistent slowness, try a smaller model from Settings → Models — SmolLM3 is the lightest option.

The output looks wrong or off-topic.

Local AI models can occasionally hallucinate, especially on very short or ambiguous selections. Try selecting a slightly larger context, or switch to a different model in Settings → Models. If a specific input consistently fails, please email us a screenshot — it helps us improve the prompts.

How do I report a bug?

Email lesminitels@protonmail.com with a description of the issue, the macOS version (Apple menu → About this Mac), the SpellFix version (SpellFix menu → About), and a screenshot if relevant.
